# helpassist

Локальный консольный ИИ-ассистент по программированию — «как Claude Code», но работает на
вашей локальной модели **qwen3** через [Ollama](https://ollama.com) и написан целиком на **Go**.
Внешних зависимостей нет — только стандартная библиотека.

## Возможности

- Интерактивный REPL в терминале с потоковым выводом ответа модели.
- Агентный цикл: модель сама вызывает инструменты, пока не решит задачу.
- Инструменты:
  | Инструмент | Действие | Подтверждение |
  |---|---|---|
  | `read_file` | прочитать файл | — |
  | `list_dir` | список каталога | — |
  | `glob` | поиск файлов по шаблону | — |
  | `grep` | поиск по содержимому (regexp) | — |
  | `write_file` | создать/перезаписать файл | да |
  | `edit_file` | точечная замена в файле | да |
  | `run_bash` | выполнить команду | да |
- Перед записью файлов и выполнением команд спрашивает подтверждение `[y/N]`.
- `Ctrl-C` прерывает текущий ход (включая запущенную команду) и возвращает к приглашению, не завершая программу.
- История диалога автоматически обрезается по старым ходам при превышении лимита, чтобы не переполнять контекст модели.

## Установка на любое устройство (Ubuntu/Debian, в т.ч. ARM/телефон)

Одной командой — скрипт сам определит архитектуру (amd64/arm64/arm), соберёт из
исходников (если есть Go) или скачает готовый бинарник из релиза, поставит в
`~/.local/bin` и проверит наличие Ollama и модели qwen3:

```sh
curl -fsSL https://raw.githubusercontent.com/ilyaswork24-byte/helpassist/main/install.sh | sh
```

После этого:
```sh
ollama pull qwen3:8b   # если модель ещё не загружена
helpassist
```

> Бинарники чистые на Go (только стандартная библиотека), без внешних
> зависимостей — поэтому одинаково работают на серверах, одноплатниках
> (Raspberry Pi) и телефонах через Termux/proot.

## Требования

- Запущенный Ollama с моделью qwen3:
  ```sh
  ollama serve              # если ещё не запущен
  ollama pull qwen3:8b
  ```
- Go ≥ 1.21 — только если собираете из исходников.

## Сборка из исходников

```sh
make build      # собрать бинарник под текущую систему
make install    # собрать и установить в ~/.local/bin
make release    # собрать бинарники под все платформы в dist/
make test       # прогнать тесты
```

Или напрямую:
```sh
go build -o helpassist .
```

## Запуск

Интерактивный режим:
```sh
./helpassist
```

Одноразовый запрос:
```sh
./helpassist "найди все TODO в проекте и покажи список"
```

## Конфигурация

| Переменная / флаг | Назначение | По умолчанию |
|---|---|---|
| `HELPASSIST_MODEL` / `--model` | модель Ollama | `qwen3:8b` |
| `OLLAMA_HOST` | адрес сервера Ollama | `http://localhost:11434` |
| `--yes` | авто-подтверждать запись и команды | выкл. |
| `--think` | включить режим рассуждений qwen3 | выкл. |
| `NO_COLOR` | отключить цвета | — |

## Команды в REPL

- `/help` — справка
- `/reset` — очистить контекст
- `/think on|off` — переключить рассуждения qwen3
- `/model [имя]` — показать/сменить модель
- `/cwd` — рабочий каталог
- `/exit` — выход (или Ctrl-D)
